


    «Пассажиравтотранс» запустил проект по внедрению в петербургские автобусы способа оплаты проезда банковскими картами. Планируется, что к концу лета данная функция будет доступна более чем на 150 маршрутах. Сейчас необходимое оборудование есть только на двух маршрутах в районе Пулкова. 

    Для внедрения функции оплаты с помощью банковских карт будет усовершенствоваться уже имеющееся оборудование. К оранжевым валидаторам у входа будет крепиться специальный стикер, к которому можно будет приложить карту. 

    Помимо оплаты банковской картой, проезд можно будет оплатить с помощью SamsungPay, ApplePay и AndroidPay.

    Система запускается совместно с Газпромбанком. С 7 июня она работает на маршрутах 39 и 39-ex, которые ходят от аэропорта до станции метро «Московская».
