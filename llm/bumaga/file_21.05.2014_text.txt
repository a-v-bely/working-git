
С 1 июня увеличится стоимость одной поездки по карточке «Подорожник». Изменения коснутся только проезда в метро, цены на проезд в наземном транспорте останутся прежними.

Стоимость жетона петербургского метрополитена составляет 28 рублей. Электронный кошелек позволяет пассажирам получать скидки на проезд: сейчас с 1-й по 10-ю поездки он стоит 27 рублей, с 11-й по 20-ю — 25,5 рубля, с 21-й по 30-ю — 23 рубля, с 31-й по 40-ю — 21,5 рубля, с 41-й — 20 рублей. Повышение цен будет неравномерным и в среднем составит 1,5 рубля.



Стоимость проезда по карточке «Подорожник»:

с 21-й по 30-ю поездку 25 рублей;

с 41-й поездки — 23 рубля.
