
Евросоюз вводит процедуру дактилоскопии для выдачи шенгенских виз. Отпечатки десяти пальцев могут начать снимать уже со второго квартала 2015 года, пишет «Российская газета» со ссылкой на Российский союз туриндустрии. 
Сбор отпечатков пальцев будет обязательным для граждан, подающих заявление в первый раз с того момента, как единая визовая информационная система (ВИС) начнет работу в конкретном регионе. Повторно проходить дактилоскопию нужно будет раз в пять лет. Кроме того, в системе будет содержаться информация о заявлениях, поданных человеком за последние пять лет, решения консульств, анкетные данные и фотография.


Ирина Тюрина, пресс-секретарь Российского союза туриндустрии:
    
— Это позволит проследить визовую историю заявителя, установить правомерность использования предыдущих виз и их действительный статус, например, при подаче заявления на многократную визу.

ВИС по всему миру будет внедряться постепенно, согласно порядку, установленному Европейской комиссией. Она же определит день, когда консульства стран Шенгенской зоны начнут реализацию проекта. Граждане с действующей шенгенской визой должны будут сдать отпечатки только при подаче следующего заявления.
