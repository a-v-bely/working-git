
Глава Facebook Марк Цукерберг заявил о первом успешном полете беспилотника Aquila для обеспечения интернетом труднодоступных регионов мира.

В дальнейшем компания хочет создать флот из нескольких Aquila, который будет летать на высоте 60 тысяч футов. Беспилотники будут связываться друг с другом с помощью лазеров и оставаться в воздухе по несколько месяцев.
